## 五、关键技术防坑（ARM 视角）

- **PMU 事件**：`LLC-load-misses` / `branch-instructions` 用 raw code（`armv8_pmuv3_0/event=0x037,.../`、`.../event=0x021,.../`），perf 4.19 不认 `pmu/符号名/` 写法。
- **钉核**：`taskset -c 0~3`（鲲鹏 920 同构 4 核，避免线程迁移噪声；无 P/E 异构）。
- **cache line**：64B（L1/L2）+ **128B（L3）** 双拐点，题1③ 曲线要标两个。
- **题1③**：必须 **pointer chasing**（绕过预取器）；朴素 `for` 遍历会被预取器抹平拐点。
- **stress-ng**：`--cpu-method list` 在 0.21+ 是语法坑（`list` 当非法 method 名），直接用 method 名即可。
- **跨边界脚本**：`ssh arm 'bash -s' <<'REMOTE'` 走 heredoc，避免引号/路径转义。
- **push 网络**：国内访问 github HTTPS 易 curl 28 断连，失败重试或换 SSH remote。
